# 📊 Vendor Performance Analysis

## 📌 Overview

This project is an end-to-end **Data Analytics** solution focused on analyzing vendor performance, inventory efficiency, pricing strategies, and profitability. The analysis combines **Python, SQL, and Power BI** to transform raw transactional data into meaningful business insights and actionable recommendations.

The project follows the complete analytics lifecycle—from data collection and cleaning to dashboard development and business reporting.

---

# 📂 Dataset

The dataset contains transactional records related to vendors, inventory, purchasing, sales, pricing, freight costs, and profitability.

### Dataset Highlights

* **10,692+ business transactions**
* Vendor information
* Brand details
* Purchase & Sales quantities
* Purchase & Selling prices
* Gross Profit & Profit Margin
* Freight Cost
* Stock Turnover
* Inventory Performance

# ▶️ How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/vendor-performance-analysis.git
```

### 2. Install Required Libraries

```bash
pip install pandas numpy matplotlib seaborn jupyter sqlalchemy psycopg2
```

### 3. Run the Jupyter Notebook

```bash
jupyter notebook
```

Open:

```
Vendor Performance Analysis.ipynb
```

### 4. SQL Analysis

* Create a database in PostgreSQL, MySQL, or SQL Server.
* Import the cleaned dataset.
* Execute SQL scripts for business analysis.

### 5. Power BI Dashboard

* Open the `.pbix` file in Power BI Desktop.
* Refresh the dataset.
* Explore interactive dashboards and KPIs.
